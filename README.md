# 📺 Bilibili Daily Dashboard 
> 专属 B站早报生成器：昨日回顾、全站热榜、AI 寄语，应有尽有。

这个项目能每天定时为您生成一份精美的 HTML 数据报表，复盘您的昨日观看足迹，并利用 AI 深度总结今日看点。

## ✨ 核心特性
1.  **多维度回顾**：昨日观看时长趋势图、历史足迹、全站及科技区热门。
2.  **动态追踪**：实时监控您关注的 UP 主（如原神、明日方舟等）近 24 小时动态。
3.  **AI 智能总结**：调用 Kimi-2.5 为您撰写风趣幽默的每日寄语。
4.  **多账号支持**：内置扫码登录管理，支持多用户快速切换。
5.  **离线快照**：每天生成独立的 HTML 和 Markdown 存档，永久保留记忆。
6.  **自动化流程**：支持 macOS 闭盖唤醒并后台生成报告。

## 🚀 快速上手 (针对新用户)

### 1. 环境准备
确保您的电脑已安装 Python 3.9+，并安装依赖：
```bash
pip install -r requirements.txt
```

### 2. 第一次运行 (初始化配置)
运行主程序，根据屏幕提示输入您的 **Kimi API Key** 和想要追踪的 **UP主 UID**：
```bash
python3 bili_daily_report.py
```
*   脚本会引导您进行 **扫码登录**。
*   配置文件会保存在 `config.yaml` 中，您可以随时手动编辑。

### 3. 设置自动运行 (仅支持 macOS)
为了每天 08:30 准时看到早报，请运行我们提供的安装脚本：
```bash
python3 install_mac_task.py
```
*   **注意**：如需在合盖状态下也准时激活，请根据脚本输出的提示开启 `pmset` 电源唤醒。

## 📂 目录结构说明
*   `daily_notes/`：按日期存放生成的 HTML 快照。
*   `data/`：存放登录凭据、观看趋势历史数据。
*   `config.yaml`：个人偏好配置。
*   `latest_report.js`：核心数据驱动文件。
