# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-03-23
### Added
- ✨ **UI 布局**：重构章节的显示逻辑为横屏滑动。
- ✨ **新功能**：支持视频抓取，优化动态和投稿的切换。
- ✨ **UI 美化与动效**：优化视频卡片 UI，新增 0.6s 段落折叠与 1s 标签切换的平滑过渡动画。
- ✨ **UI 美化与动效**：侧边栏增加章节可见性开关，增加关注up主目录，支持跳转。
- ✨ **信息源切换**：可切换同一天内生成的不同报告显示。
- 🖼 **多图动态网格**：动态抓取升级支持最高 9 张图片，并实现响应式网格布局。
- ✨ **任务定时**：引入mac端自动任务设定创建程序。
- 🔍 **交互式看图 (Lightbox)**：新增零依赖全屏图片查看器，支持一键放大、背景模糊及 ESC 快捷退出。
- 📝 **长内容支持**：将动态抓取上限提升至 2000 字符，并优化了配套的滚动条样式与容器高度。
- ⚡️ **实时进度输出**：在终端增加带时间戳和 Emoji 的执行进度条。
- 🔭 **Dashboard 资源过滤**：看板侧边栏新增“信源控制”面板。


## [0.1.0] - 2026-03-21
### Added
- 🎉 初版发布，支持基本的 B 站日报生成与可视化功能。
- 🤖 集成 Kimi AI API 用于数据摘要。
- 📈 支持基础观看时长趋势图表展示.
